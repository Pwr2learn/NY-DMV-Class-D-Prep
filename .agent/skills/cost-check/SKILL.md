# Steps
1. Identify cost drivers.
2. Choose smaller model/scope where safe.
3. Batch or cache repeated work.
4. Prefer local/static checks before paid/external calls.
5. Define stop condition.

# Output Format
```md
Cost drivers:
Cheaper safe path:
Approval needed:
Stop condition:
```
