# Steps
1. Define research question.
2. Gather sources.
3. Separate facts, assumptions, and recommendations.
4. Note uncertainty.
5. Provide next step.

# Output Format
```md
Summary:
Key findings:
Sources/evidence:
Uncertainty:
Recommended next step:
```
