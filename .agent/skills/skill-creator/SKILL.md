---
name: skill-creator
description: Use when a repeated workflow occurs 2+ times or a durable process should become reusable.
---


# Steps
1. Identify repeatable workflow.
2. Define trigger description.
3. Write compact steps.
4. Add stop conditions.
5. Add validation.
6. Add gotchas.
7. Add output format.

# Output Format
Create a `SKILL.md` with:
```md
---
name:
description:
---
# Steps
# Stop Conditions
# Validation
# Gotchas
# Output Format
```
