# AI Build Prompt

This file contains the original build prompt used to initialize the NY DMV Class D Prep application.

## Core Directives
1. Build a bilingual web app (English and Spanish) to help users prepare for the NY DMV Class D learner permit test.
2. Ground all test content exclusively in the NY DMV manual.
3. Test users until they are highly confident they can pass the real DMV test.
4. Support Learn Mode, Practice Mode, and Mock Test Mode.
5. Provide strict progress tracking and result summaries.

## Rules & Standards
- Keep documentation up to date (PROJECT_BLUEPRINT.md is the source of truth).
- Prioritize Git-friendly changes.
- Implement robust accessibility features (dark/light mode, font sizes).
- Mock Test grading must match the exact NY DMV passing rules (14/20 overall, 2/4 road signs).

## Original Input Summary
The original input detailed an exhaustive list of 20 points, covering Documentation, Git, Core Features, Language requirements, Accessibility, Official NY DMV logic, Difficulty Modes, Experiences (Mock, Practice, Learn), Results, Progress Tracking, Question Bank rules, Quiz Engine, Main Pages, Build Order, MVP guidelines, Testing, and Deployment Readiness.
