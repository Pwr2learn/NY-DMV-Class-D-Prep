# START HERE

## Quick Start

1. Rename one of the project folders.
2. Open that project folder in Antigravity.
3. Fill out `GEMINI.md`.
4. Ask Gemini 3.1 Pro to read `GEMINI.md` and `.agent/RESOURCE_REGISTRY.md` first.
5. Use `/validate` before implementation.
6. For websites, use the relevant design skill.
7. Before launch, use `/pre-launch-security-check`.
